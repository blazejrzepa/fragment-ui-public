# @fragment_ui/ui

React component library for Fragment UI Design System.

## Installation

```bash
pnpm add @fragment_ui/ui @fragment_ui/tokens
```

## Quick Start

```tsx
import { Button } from "@fragment_ui/ui";

export default function MyPage() {
  return <Button>Click me</Button>;
}
```

## Install Components via Registry

You can install individual components using the registry:

```bash
# Using shadcn CLI
npx shadcn@latest add https://fragmentui.com/r/button.json

# Or use the Fragment UI CLI
npx @fragment-ui/cli add button
```
