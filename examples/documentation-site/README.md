# Documentation Site Example

A complete documentation site built with Fragment UI Documentation blocks.

## Features

- **DocumentationHeader** - Fixed header with backdrop blur, navigation links, search, and actions
- **DocumentationSidebar** - Left sidebar navigation with scroll detection and wheel handling
- **DocumentationLayout** - Complete layout with auto-extracted TableOfContents
- Responsive design (mobile/desktop)
- Dark mode support
- Active state detection
- Smooth scrolling

## Get Started

```bash
cd documentation-site
pnpm install
pnpm dev
```

Open [http://localhost:3002](http://localhost:3002)

## What's Included

This example demonstrates:

1. **Complete Documentation Layout**
   - Header with logo, navigation, search, and theme toggle
   - Sidebar with hierarchical navigation
   - Main content area with auto-extracted TableOfContents
   - Right sidebar for additional content

2. **Navigation Structure**
   - Get Started section
   - Components documentation
   - Blocks documentation
   - Guides and examples

3. **Best Practices**
   - Using Documentation blocks together
   - Configuring navigation sections
   - Customizing header and sidebar
   - Integrating with Next.js routing
