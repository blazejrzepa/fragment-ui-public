# Fragment UI Examples

This directory contains example applications demonstrating how to use Fragment UI in real-world scenarios.

## Available Examples

### 1. Next.js Dashboard (`nextjs-dashboard`)

A complete dashboard application built with Next.js and Fragment UI blocks.

**Features:**
- Dashboard layout with navigation
- KPI cards and metrics
- Data visualization widgets
- Responsive design

**Get Started:**
```bash
cd nextjs-dashboard
pnpm install
pnpm dev
```

Open [http://localhost:3000](http://localhost:3000)

### 2. SaaS Settings (`saas-settings`)

A SaaS application settings page demonstrating forms, data tables, and navigation patterns.

**Features:**
- Settings screen with multiple sections
- Form containers and inputs
- Billing and subscription management
- Security settings

**Get Started:**
```bash
cd saas-settings
pnpm install
pnpm dev
```

Open [http://localhost:3001](http://localhost:3001)

### 3. Documentation Site (`documentation-site`)

A complete documentation site built with Fragment UI Documentation blocks.

**Features:**
- DocumentationHeader with backdrop blur and navigation
- DocumentationSidebar with scroll detection
- DocumentationLayout with auto-extracted TableOfContents
- Responsive design
- Dark mode support

**Get Started:**
```bash
cd documentation-site
pnpm install
pnpm dev
```

Open [http://localhost:3002](http://localhost:3002)

## Installation

Each example includes its own `package.json` with Fragment UI dependencies. To use Fragment UI in your own project:

```bash
# Install Fragment UI packages
pnpm add @fragment_ui/ui @fragment_ui/tokens @fragment_ui/blocks
```
