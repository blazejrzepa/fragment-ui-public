Fragment UI is an **AI-native design system** based on shadcn/ui with **code-first distribution**. It combines:

- **63+ Production-Ready Components** - Built on Radix UI primitives
- **Code-First Distribution** - Install components via registry (`shadcn add`)
- **AI-Powered Studio** - Generate and edit UI through conversational interface
- **Enterprise Tooling** - CLI, VS Code extension, MCP server, telemetry
